1. Что означает RDD в Apache Spark?
- Resilient Distributed Dataset - отказоустойчивый распределенный набор данных

2. Какая разница между трансформацией и действием?
- Трансформация создает новый DataFrame, действие возвращает результат в Python

3. Что такое lazy evaluation и зачем она нужна?
- Вычисления откладываются до вызова действия, позволяет оптимизировать план

4. Какой класс является основной точкой входа в PySpark?
- SparkSession - начиная с Spark 2.0

5. Что такое partition в Spark?
- Логический раздел данных для распределенной обработки

6. Какие преимущества DataFrame по сравнению с RDD?
- Лучшая производительность за счет Catalyst Optimizer, строгая типизация

7. Что делает Catalyst Optimizer?
- Оптимизирует SQL и DataFrame запросы

8. Какие основные компоненты архитектуры Spark вы знаете?
- Driver, Cluster Manager, Executors, Worker Nodes

9. Что такое DAG в контексте Spark?
- Directed Acyclic Graph - граф выполнения задач без циклов

10. Какие форматы данных наиболее эффективны в Spark?
- Parquet, ORC, Delta Lake - столбчатые форматы с метаданными

11. Что такое SparkSession и зачем он нужен?
- Единая точка входа для работы с DataFrame, SQL, Streaming

12. Какие настройки конфигурации наиболее важны?
- spark.executor.memory, spark.sql.adaptive.enabled, spark.driver.memory

13. Что такое сериализация в Spark?
- Преобразование объектов в байты для передачи по сети (Kryo serializer)

14. Как Spark обеспечивает отказоустойчивость?
-  Через lineage graph - восстановление по логу операций

15. Что такое executor в архитектуре Spark?
- Процесс на рабочем узле, выполняющий вычисления

16. Как работает кэширование в Spark?
- Сохранение данных в памяти/диску для быстрого доступа

17. Что такое broadcast variable?
- Переменные, эффективно рассылаемые всем узлам

18. Какие типы данных поддерживаются в DataFrame?
- Поддерживаются все типы данных Spark SQL (String, Integer, Struct и др.)

19. Что такое стадии (stages) в Spark?
- Логические блоки выполнения, состоящие из задач

20. Какие best practices по оптимизации Spark приложений вы знаете?
- Правильное партиционирование, кэширование, оптимальные форматы данных
